Crypto Portfolio Dashboard

This project demonstrates how business intelligence tools can be used to monitor and analyse cryptocurrency markets in near real time. The dashboard was built in Power BI using live data sourced from the CoinGecko API.

Tools and Technologies

  • Power BI Desktop for interactive reporting
  • Power Query for data cleansing and preparation
  • CoinGecko API as a real-time data source

Business Questions Addressed

  • Which assets have the highest market capitalisation?
  • Which cryptocurrencies are gaining or losing value?
  • How volatile is the market on a daily basis?

Dashboard Capabilities

  • High-level market overview
  • Performance comparison across assets
  • User-driven filtering and exploration
